About This Project

Understanding Malaysia's Job Market at Scale

InsightMY Analysis processes rich Kaggle datasets to surface clear, data-backed insights on hiring patterns, salary distributions, skill demands, and industry growth — presented through an interactive Power BI dashboard.

Big Data Analysis Interactive Visuals Kaggle Datasets Malaysia-Focused
Data Collection
Sourced structured job listing data from comprehensive Kaggle datasets covering all major Malaysian states.
Data Processing
Cleaned, transformed, and enriched the raw data using Python pipelines to ensure accuracy and analysis-readiness.
Dashboard Insights
Visualised key findings in Power BI with filters for state, industry, salary range, and job type.
Actionable Results
Surface hiring trends, top skills, and salary benchmarks to help job seekers and researchers alike.

# InsightMY Analysis

A Big Data-driven job market analytics platform that transforms Malaysian job listing datasets into actionable insights through interactive dashboards and business intelligence reporting.

## Live Demo
https://insight-my.onrender.com

## Project Overview
InsightMY Analysis analyzes Malaysian job market data to identify:

- Hiring trends across states
- Salary benchmarks
- Industry demand
- In-demand skills
- Job category distribution
